# ExamsTeam

## О проекте

`ExamsTeam` — это многостраничный фронтенд-проект медицинского центра, сверстанный на `HTML` и `Tailwind CSS`.
Проект включает главную страницу и несколько внутренних страниц направлений/услуг, собранных по референсам и локальным изображениям из папки `IMage`.

Главная цель проекта:

- собрать аккуратный статический сайт медицинского центра;
- связать страницы между собой через навигацию;
- оформить страницы по секциям;
- использовать локальные изображения и готовые SVG-иконки;
- получить готовую структуру для демонстрации, защиты или дальнейшей доработки.

## Что реализовано

- главная страница [index.html](./index.html);
- внутренние страницы:
  - [Pages/munis.html](./Pages/munis.html)
  - [Pages/behruz.html](./Pages/behruz.html)
  - [Pages/abdullo.html](./Pages/abdullo.html)
- навигация между страницами `Page 1-4`;
- якорные переходы по секциям внутри страниц;
- секционная верстка:
  - hero-блоки;
  - карточки услуг;
  - формы записи;
  - блоки врачей;
  - отзывы;
  - лицензии и сертификаты;
  - карты и футеры;
- использование локальных изображений и иконок из папки `IMage`;
- сборка стилей через `Tailwind CSS CLI`.

## Технологии

- `HTML5`
- `Tailwind CSS v4`
- `Tailwind CLI`
- локальные SVG, PNG и JPG-ассеты

## Как запустить проект

### 1. Установить зависимости

```bash
npm install
```

### 2. Собрать CSS

Обычная сборка:

```bash
npx @tailwindcss/cli -i ./src/input.css -o ./src/output.css
```

Режим отслеживания изменений:

```bash
npx @tailwindcss/cli -i ./src/input.css -o ./src/output.css --watch
```

### 3. Открыть проект

Можно открыть [index.html](./index.html) напрямую в браузере или через `Live Server` в редакторе кода.

## Особенности проекта

- Проект не использует JavaScript-фреймворки и собран как статический сайт.
- Основной упор сделан на верстку, визуальное сходство с макетами и работу с локальными ассетами.
- Формы в проекте — фронтенд-блоки без подключенного backend.
- Некоторые изображения и повторяющиеся UI-компоненты распределены по разным папкам внутри `IMage`.
