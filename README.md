# ExamsTeam

## О проекте

`ExamsTeam` — это многостраничный учебный frontend-проект медицинского центра, собранный на `HTML` и `Tailwind CSS`.
В проекте реализованы главная страница и несколько внутренних страниц направлений, каждая из которых оформлена по секциям и связана общей навигацией.

Проект создавался как командная работа по верстке:

- с ручной HTML-структурой;
- с использованием `Tailwind CSS`;
- с подключением локальных изображений и иконок;
- с переходами между страницами и якорями внутри страниц.

## Стек технологий

- `HTML5`
- `Tailwind CSS v4`
- `@tailwindcss/cli`
- локальные `SVG`, `PNG`, `JPG` ассеты

## Как запустить проект

### 1. Установить зависимости

```bash
npm install
```

### 2. Собрать стили

```bash
npx @tailwindcss/cli -i ./src/input.css -o ./src/output.css
```

### 3. Открыть проект

Откройте [index.html](./index.html) в браузере или используйте `Live Server`.

## Особенности проекта

- Проект статический и не использует JavaScript-фреймворки.
- Основной фокус сделан на точную секционную верстку.
- Все страницы используют локальные изображения из папки `IMage`.
- Формы в проекте — это frontend-часть без подключенного backend.
